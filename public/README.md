# Public Folder - Static Assets

This folder contains static assets that are served directly by Next.js.

## Resume PDF

**To add your resume:**
1. Place your resume PDF file in this folder (`public/`)
2. Name it exactly: `resume.pdf`
3. The file will be accessible at `/resume.pdf` in your website
