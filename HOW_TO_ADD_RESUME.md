# How to Add Your Resume PDF

## Quick Steps

1. **Find your resume PDF file** on your computer
2. **Copy it** to the `public/` folder in this project
3. **Rename it** to exactly: `resume.pdf`
4. **Test it** by clicking the download buttons on your website

## File Location

```
Portfolio/
└── public/
    └── resume.pdf  ← Put your PDF file here
```

## Where to Find Your PDF

If you've already shared a PDF file with me, check these common locations:

1. **Downloads folder**: `C:\Users\BHARAT\Downloads\`
2. **Desktop**: `C:\Users\BHARAT\Desktop\`
3. **OneDrive**: `C:\Users\BHARAT\OneDrive - Pace University\`
4. **Documents**: `C:\Users\BHARAT\Documents\`

## Verification

After adding the file, verify:
- ✅ File is named exactly `resume.pdf` (lowercase)
- ✅ File is in the `public/` folder (not `public/images/` or anywhere else)
- ✅ File opens correctly when you double-click it

## Testing

1. Start your development server: `npm run dev`
2. Visit `http://localhost:3000`
3. Click the "Download Resume" button
4. The PDF should download or open in a new tab

## Need Help?

If you can't find your resume PDF:
- Check your email attachments
- Look in your cloud storage (OneDrive, Google Drive, etc.)
- Check if you have it saved with a different name

Once you find it, just copy it to `public/resume.pdf` and you're done!
